# Matrix Multiplication
This code implements the matrix multiplication algorithm.

## Example
```
[1, 1]     [1, 2]     [1 * 1 + 1 * 2, 1 * 2 + 1 * 2]     [3, 4]
[3, 4]  X  [2, 2]  =  [3 * 1 + 4 * 2, 3 * 2 + 4 * 2]  =  [11, 14]
```
